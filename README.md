# Pakistan Universities Directory Web Application

## Overview
Pakistan Universities Directory is a responsive web application that allows users to browse and search universities across Pakistan. The application displays university information in a card-based layout, including university logos, contact numbers, and official website links.

## Features
- Search universities by name
- Responsive card-based user interface
- University logo display
- Contact information display
- Direct links to official university websites
- Pagination support
- Clean and user-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Responsive Web Design

## How to Run

1. Clone the repository
2. Open the project folder
3. Run the application using Live Server or any local server
4. Open in your browser

## Functionality

- Users can search universities using the search bar.
- University cards display:
  - University Name
  - University Logo
  - Contact Number
  - Official Website Link
- Pagination helps users navigate through multiple university records.
